# 回答格式模板

## 概念讲解

```
📖 [主题名称]

▸ 核心概念
  [用简洁的语言解释核心概念]

▸ 通俗理解
  [用类比或生活化的例子帮助理解]

▸ 关键要点
  - 要点1
  - 要点2
  - 要点3

▸ 示例
  [具体的例子来展示概念的应用]

▸ 你来试试
  [要求用户用自己的话复述或回答一个应用性问题]
```

## 技术知识讲解

在概念讲解格式基础上，额外提供：
- 代码示例（如果适用）
- 实际应用场景
- 常见误区和注意事项
- 与相关技术的对比

## 回答问题

- 先直接回答问题
- 然后解释为什么是这样
- 最后补充相关的背景知识

## 后测问题格式

```
🧪 检验时间

Q1: [生成式问题 — 要求用自己的话解释]
Q2: [应用式问题 — 将概念应用到新场景]
Q3: [对比式问题 — 比较两个相关概念的异同]
```

## 学习效果可视化（阶段四结束时输出）

每次学习结束时，必须输出以下格式的效果总结：

```
📊 本次学习效果

知识点          等级变化       状态
─────────────────────────────────────
列表推导式      ⬆ 2 → 4      ✅ 掌握
装饰器          ⬆ 1 → 3      📈 进步中
生成器          ── 1 → 1      🔄 需重点复习

整体正确率：前测 40% → 后测 80%
本次新掌握：2 个知识点
下次复习：2025-02-01（装饰器、生成器）
```

等级变化符号说明：
- `⬆` 等级提升
- `⬇` 等级下降（复习时遗忘）
- `──` 等级不变

状态标签：
- ✅ 等级 4-5，已掌握
- 📈 等级 3，进步中
- 🔄 等级 1-2，需重点复习

## 复习报告模板（快捷复习模式结束时输出）

```
📋 复习报告

复习知识点：{N} 个（跨 {M} 个领域）

知识点          领域        结果     等级变化
──────────────────────────────────────────────
装饰器          Python      ✅ 正确   ⬆ 3 → 4
梯度下降        机器学习     ❌ 错误   ⬇ 3 → 2
闭包            Python      ✅ 正确   ⬆ 3 → 4

正确率：67%（2/3）
下次最近复习：2025-02-05（梯度下降）
```
